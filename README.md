# Mastermind

A modern, full-featured React implementation of the classic Mastermind logic game.

## Features

- **Three Game Modes**:
  - **Single Player**: Challenge the computer and crack its randomly generated code.
  - **Two Player Duel**: Play locally with a friend. One sets the code, the other breaks it, with built-in privacy gates and manual scoring.
  - **AI Challenge**: You set the code and watch the AI employ logical deduction to solve it in real-time.
- **Classic Mechanics**: 4-peg codes, 6 color options, and 10 attempts to win.
- **Modern UI**: Built with React, Tailwind CSS v4, and Lucide Icons for a sleek, responsive experience.
- **Visual Feedback**: Includes victory confetti, intuitive feedback pegs, and a clear game board.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
